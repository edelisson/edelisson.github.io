I made this website as a personal hub to act as an alternative to social media.

This took some refamiliarizing with HTML as well as familiarizing with CSS, but all things considered I am still a novice in both of them.
For learning, I have used w3schools, Rolf Staflins "HTML- & CSS-boken" (the HTML- and CSS-book) as well as a number of Youtube-videos.
There are some technical errors in the source that I intend to fix, once the know-how is sharpened up.

Because my field of study (archival science) generally encourages the use of open source software (for reasons I will not go into now),
and because I ideologically agree with it (for the most part), I run FOSS when possible. For this website I've used ATOM,
and the laptop I'm using has Linux Mint. There is a blog post here about my migration to primarily FOSS.
